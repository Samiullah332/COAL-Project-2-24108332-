# COAL-Project-2-24108332-
Group project developed for the Computer Organization and Assembly Language (COAL) Lab. This project demonstrates binary-to-hexadecimal data conversion in 8086 Assembly Language using register manipulation, logical operations, and ASCII conversion techniques in the emu8086 simulator.
